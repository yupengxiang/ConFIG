### Q4: 计算凸函数在指定点的次微分 (Subdifferential)
**次微分定义**：对于凸函数 $f(x)$，点 $x_0$ 处的次微分 $\partial f(x_0)$ 是所有满足 $f(x) \ge f(x_0) + g^T(x - x_0), \forall x$ 的次梯度 $g$ 的集合。

**（a）$f(x_1, x_2, x_3) = \max(|x_1|, |x_2|, |x_3|)$，在点 $(0,0,0)$**
*   **解答**：该函数本质上是 $L_\infty$ 范数，即 $f(x) = \|x\|_\infty$。
*   根据次微分性质，$L_\infty$ 范数在原点的次微分是其对偶范数（$L_1$ 范数）的单位球。
*   具体推导：绝对值函数在 0 点的次微分为 $\partial |x_i|(0) =[-1, 1]$。最大值函数在所有分支均激活（均为0）时的次微分，是所有激活分支次梯度的凸包。
*   **结论**：$\partial f(0,0,0) = \{ (g_1, g_2, g_3)^T \in \mathbb{R}^3 \mid |g_1| + |g_2| + |g_3| \le 1 \}$。（即 $L_1$ 单位球）。

**（b）$f(x) = e^{|x|}$，在点 $x=0$**
*   **解答**：设 $h(y) = e^y$，$g(x) = |x|$。因为 $h(y)$ 在 $\mathbb{R}$ 上单调递增且可导，$f(x) = h(g(x))$。
*   根据次微分链式法则：$\partial f(x) = h'(g(x)) \partial g(x)$。
*   在 $x=0$ 处，$g(0) = 0$，$h'(0) = e^0 = 1$。而 $\partial g(0) =[-1, 1]$。
*   **结论**：$\partial f(0) = 1 \times[-1, 1] = [-1, 1]$。

**（c）$f(x_1, x_2) = \max\{x_1+x_2-1, x_1-x_2+1\}$，在点 $(1,1)$**
*   **解答**：设 $f_1(x) = x_1+x_2-1$，$f_2(x) = x_1-x_2+1$。
*   在点 $(1,1)$ 处，$f_1(1,1) = 1+1-1=1$，$f_2(1,1) = 1-1+1=1$。两者函数值相等，说明两个分支都处于**激活状态**。
*   求各分支梯度：$\nabla f_1(1,1) = (1, 1)^T$，$\nabla f_2(1,1) = (1, -1)^T$。
*   最大值函数的次微分是所有激活分支梯度的**凸包（Convex Hull）**。
*   **结论**：$\partial f(1,1) = \text{Conv}\{ (1,1)^T, (1,-1)^T \} = \left\{ \begin{pmatrix} 1 \\ 2\alpha - 1 \end{pmatrix} \bigg| \alpha \in [0, 1] \right\}$。

### Q8: 带噪声的随机梯度更新收敛性证明
**题目**：$x_{t+1} = x_t - \lambda(\nabla f(x_t) + \sigma \xi_t)$，其中 $\xi_t \sim N(0,1)$，$f(x) = \frac{1}{2}\|x\|_2^2$。证明 $x_t$ 的收敛分布并解释影响。
*   **证明**：
    已知 $\nabla f(x_t) = x_t$，代入迭代公式得：
    $$x_{t+1} = x_t - \lambda(x_t + \sigma \xi_t) = (1-\lambda)x_t - \lambda \sigma \xi_t$$
    因为 $\xi_t \sim \mathcal{N}(0,1)$ 且独立同分布，这是一个典型的 **AR(1) 过程（自回归过程）**。
    1. **求期望**：$\mathbb{E}[x_{t+1}] = (1-\lambda)\mathbb{E}[x_t]$。由于 $\lambda < 1$（通常假定 $0 < \lambda < 1$），当 $t \to \infty$ 时，$\mathbb{E}[x_t] \to 0$。
    2. **求方差**：对方程两边取方差，因为 $x_t$ 与 $\xi_t$ 独立：
       $$\text{Var}(x_{t+1}) = (1-\lambda)^2 \text{Var}(x_t) + \lambda^2 \sigma^2 \text{Var}(\xi_t)$$
       设稳态方差为 $V_{\infty}$，令 $t \to \infty$ 有 $\text{Var}(x_{t+1}) = \text{Var}(x_t) = V_{\infty}$：
       $$V_{\infty} = (1-\lambda)^2 V_{\infty} + \lambda^2 \sigma^2$$
       $$V_{\infty} - (1 - 2\lambda + \lambda^2)V_{\infty} = \lambda^2 \sigma^2 \implies (2\lambda - \lambda^2)V_{\infty} = \lambda^2 \sigma^2$$
       $$V_{\infty} = \frac{\lambda^2 \sigma^2}{2\lambda - \lambda^2}$$
    *   **结论**：由于线性组合的传递性，$x_t$ 服从正态分布，故 $t \to \infty$ 时，$x_t \sim \mathcal{N}\left(0, \frac{\lambda^2 \sigma^2}{2\lambda - \lambda^2}\right)$，得证。
*   **注入噪声的影响（物理解释）**：
    1. **无法精确定位极小值**：由于方差的存在，算法在收敛后会在极小值附近进行“布朗运动”（随机游走），而无法停留在绝对的 0 点。
    2. **正则化与逃离鞍点（极其重要）**：在深度学习的非凸优化中，这种稳态噪声能帮助模型跃出尖锐的极小值（Sharp Minima）或鞍点，进入平缓的极小值（Flat Minima），从而获得**更强的泛化能力（Generalization）**。

### Q10: 梯度下降与动量加速的收敛性简证
**1. 梯度下降法（目标函数为凸函数）**
*   假设 $f(x)$ 是 $L$-Lipschitz 平滑的凸函数，步长 $\alpha \le \frac{1}{L}$。
*   由平滑性引理：$f(x_{k+1}) \le f(x_k) + \nabla f(x_k)^T(x_{k+1} - x_k) + \frac{L}{2}\|x_{k+1} - x_k\|^2$。代入 $x_{k+1} = x_k - \alpha \nabla f(x_k)$ 得：
    $$ f(x_{k+1}) \le f(x_k) - \frac{\alpha}{2}\|\nabla f(x_k)\|^2 $$
*   结合凸性 $f(x^*) \ge f(x_k) + \nabla f(x_k)^T(x^* - x_k)$，经过放缩和求和（Tele-scoping），可证得：
    $$ f(x_k) - f(x^*) \le \frac{\|x_0 - x^*\|^2}{2\alpha k} $$
    即凸函数下，梯度下降具有 $O(1/k)$ 的次线性收敛率。若为**强凸函数**（参数为 $\mu$），则可通过压缩映射原理证得 $\|x_k - x^*\|^2 \le (1-\alpha\mu)^k\|x_0 - x^*\|^2$，即 $O(e^{-ck})$ 的**线性收敛率**。

**2. 动量加速法（Nesterov Accelerated Gradient, NAG）**
*   证明思想：普通 GD 严重依赖条件数 $\kappa = L/\mu$。Nesterov 引入了**势函数（Lyapunov Function/Estimate Sequence）**机制。
*   通过构造势函数 $\Phi_k = a_k(f(x_k) - f^*) + b_k\|z_k - x^*\|^2$，并证明 $\Phi_{k+1} \le \Phi_k$。
*   结果：NAG 能够将凸函数的收敛率从 $O(1/k)$ 加速到 $\mathbf{O(1/k^2)}$；将强凸函数的收敛率从依赖条件数 $(1-1/\kappa)^k$ 优化到依赖条件数的平方根 $(1-1/\sqrt{\kappa})^k$，这是目前一阶优化方法能达到的理论**最优下界（Optimal Lower Bound）**。

---

### Q17: 共轭函数的定义、计算与对偶性的联系
*   **定义**：函数 $f(x)$ 的 Fenchel 共轭函数 $f^*(y)$ 定义为：
    $$ f^*(y) = \sup_{x \in \text{dom} f} \left( y^T x - f(x) \right) $$
*   **如何求解**：求解共轭函数实质上是求关于 $x$ 的极值。令导数 $\nabla_x (y^T x - f(x)) = y - \nabla f(x) = 0$，即解出使得 $\nabla f(x) = y$ 的 $x$，然后再将其代回原式即可求得 $f^*(y)$。
*   **与对偶性的联系**：拉格朗日对偶问题的推导，其核心步骤本质上就是在**求解共轭函数**。比如对于带等式约束的优化问题，引入拉格朗日乘子 $\nu$ 后求下确界，$\inf_x (f(x) + \nu^T(Ax-b)) = -b^T\nu - \sup_x (-A^T\nu x - f(x)) = -b^T\nu - f^*(-A^T\nu)$。共轭函数是将复杂的原问题转换为凸的对偶问题的桥梁。

# 模块二：高阶理论与文献简答 (Q9, 11, 12, 15, 16)

**Q9：两次随机初始化的神经网络参数 $\theta_1$ 和 $\theta_2$ 相似吗？为什么？**
*   **不相似（在参数空间中）**：因为神经网络的损失函数极度非凸，存在大量的局部极小值。此外，神经网络具有**“排列对称性”（Permutation Symmetry）**，即打乱隐藏层神经元的顺序，网络的数学表达完全不变，但参数矩阵彻底改变。因此 $\theta_1$ 和 $\theta_2$ 在欧氏空间中距离极远。
*   **但它们在功能上相似（函数空间/预测空间中）**：近年来关于**“模式连接性”（Mode Connectivity）**的前沿研究表明，尽管 $\theta_1$ 和 $\theta_2$ 在空间中相距甚远，但通常可以找到一条平滑的非线性路径（或低维子空间）将它们连起来，且路径上的 Loss 始终很低。

**Q11：转化思想总结**
*   **非凸转凸**：松弛变量法（Relaxation，如整数规划松弛为线性规划，二次非凸松弛为半正定规划 SDP）、凸包代替（Convex Envelope）、对偶转换（弱对偶性转换为下界）。
*   **有约束转无约束**：罚函数法（外点法/内点罚）、障碍函数法（内点法）、拉格朗日乘子法、**增广拉格朗日法/ADMM**（解决罚函数法病态条件数的问题）。

**Q12：最速下降、牛顿、修正牛顿的统一描述与变尺度思想**
*   **统一公式**：$x_{k+1} = x_k - \alpha_k B_k \nabla f(x_k)$
    *   $B_k = I \implies$ **最速下降法**（只用一阶信息，易Zig-Zag）。
    *   $B_k = [\nabla^2 f(x_k)]^{-1} \implies$ **牛顿法**（用真实海森逆矩阵，二次收敛，但不保证下降方向，且计算量巨大）。
    *   $B_k = [\nabla^2 f(x_k) + \mu I]^{-1} \implies$ **修正牛顿法**（加对角正则化，保证正定性和下降方向）。
*   **变尺度法（拟牛顿法）的基本思想**：本质是空间变换。因为普通梯度在椭圆形的等高线中会严重震荡，变尺度法通过迭代构造一个近似海森逆矩阵 $B_k$，它相当于**将空间的坐标轴进行了拉伸和旋转（仿射变换）**，把原本扁长的椭圆等高线变成正圆形，从而让一阶梯度直接指向圆心（极小值）。

**Q15：一阶优化算法的最新加速算法思想（林宙辰老师书籍）**
*   林宙辰老师在《Accelerated Optimization for Machine Learning》中总结的加速思路核心包括：
    1.  **外推法（Extrapolation / Momentum）**：以 Nesterov 加速为代表，利用历史动量纠正当前梯度，抑制震荡。
    2.  **近端梯度（Proximal Gradient）**：针对包含非平滑项的优化问题（如L1），采用 FISTA 等加速算法进行分裂求解。
    3.  **方差缩减（Variance Reduction）**：在随机优化中，通过 SVRG、SAGA 等思想，利用全梯度信息修正随机梯度，让 SGD 也能达到线性收敛。

**Q16：Krylov 子空间方法与 GMRES**
*   **基本思想**：当求解超大维度的线性方程组 $Ax=b$ 时，无法直接求逆。Krylov 方法通过不断对向量乘矩阵，构造出一个降维的子空间 $\mathcal{K}_n = \text{span}\{b, Ab, A^2b, \dots, A^{n-1}b\}$，然后在这个小空间里寻找最优近似解，极大降低了计算复杂度。
*   **与 CG 的异同**：
    *   **共轭梯度法 (CG)** 只能用于 $A$ 是**对称正定矩阵 (SPD)** 的情况。
    *   **广义最小残差法 (GMRES)** 不要求矩阵对称，也不要求正定，可以求解**非对称的一般大稀疏方阵**，但代价是每次迭代需要存储正交基，内存消耗比 CG 大（所以通常采用截断版 Restarted GMRES）。

[

其中：

\begin{pmatrix}
4 & 2\
2 & 2
\end{pmatrix},
\quad
b=
\begin{pmatrix}
1\
-1
\end{pmatrix}.

[
\nabla f(x)=Ax+b.
]

最优点满足：

[
Ax^*+b=0.
]

所以：

[
x^*=-A^{-1}b
============

\begin{pmatrix}
-1\
\frac32
\end{pmatrix}.

\alpha_0
========

-\frac{g_0^Td_0}{d_0^TAd_0}

## Q3：不精确一维搜索

Rosenbrock 函数：

[
f(x)=100(x_2-x_1^2)^2+(1-x_1)^2.
]

给定：

[
x^{(0)}=(-1,1)^T,
\quad
d=(1,1)^T.
]

令：

[
\phi(\lambda)=f(x^{(0)}+\lambda d).
]

则：

[
x_1=-1+\lambda,
\quad
x_2=1+\lambda.
]

代入得到：

[
\phi(\lambda)
=============

100\lambda^4-600\lambda^3+901\lambda^2-4\lambda+4.
]

[
\phi'(0)=-4<0.
]

所以 (d) 是下降方向。

这里有一个值得写进报告的点：
(\phi(\lambda)) 是四次函数，沿这个方向可能存在多个驻点。不精确线搜索的目的不是找全局最优 (\lambda)，而是找一个满足充分下降条件和曲率条件的可接受步长。

因此 Q3 的报告建议重点比较：

* Armijo：只要求充分下降；
* Goldstein：要求下降不能太小也不能太大；
* Wolfe-Powell：同时控制充分下降和曲率；
* 改进 Wolfe-Powell：更强调数值稳定性。

## Q8：带噪声 SGD

建议完整写成：

因为：

[
f(x)=\frac12 |x|_2^2,
]

所以：

[
\nabla f(x)=x.
]

代入迭代式：

[
x_{t+1}
=======

# x_t-\lambda(x_t+\sigma\xi_t)

(1-\lambda)x_t-\lambda\sigma\xi_t.
]

这是 AR(1) 递推。

期望满足：

[
\mathbb E[x_{t+1}]
==================

(1-\lambda)\mathbb E[x_t].
]

当 (0<\lambda<1) 时：

[
\mathbb E[x_t]\to 0.
]

方差满足：

[
\operatorname{Var}(x_{t+1})
===========================

(1-\lambda)^2\operatorname{Var}(x_t)
+
\lambda^2\sigma^2.
]

设平稳方差为 (V)，则：

[
V=(1-\lambda)^2V+\lambda^2\sigma^2.
]

所以：

[
V[1-(1-\lambda)^2]=\lambda^2\sigma^2.
]

即：

[
V(2\lambda-\lambda^2)=\lambda^2\sigma^2.
]

因此：

[
V=
\frac{\lambda^2\sigma^2}{2\lambda-\lambda^2}.
]

所以：

[
x_t
\Rightarrow
\mathcal N
\left(
0,
\frac{\lambda^2\sigma^2}{2\lambda-\lambda^2}
\right).
]

解释：

> 噪声使迭代点不能精确停在极小值，而是在极小值附近形成一个平稳分布。噪声越大，平稳方差越大；步长越大，震荡越明显。在非凸优化中，噪声有时可以帮助算法逃离鞍点或尖锐局部极小值，但也会降低最终定位精度。

## Q9：两次神经网络训练得到的参数是否相似？

建议答案：

> 一般来说，(\theta_1) 和 (\theta_2) 在参数空间中不一定相似，但在函数空间中可能相似。

原因：

1. 神经网络优化问题高度非凸；
2. 不同随机初始化会进入不同的低损失区域；
3. 网络存在排列对称性，例如交换隐藏层神经元顺序，函数不变但参数完全不同；
4. 过参数化网络中存在大量等价或近似等价解；
5. 因此参数向量的欧氏距离可能很大，但模型预测效果、训练损失和测试性能可能接近。

可以加一句提高质量：

> 因此，比较两个网络是否“相似”时，直接比较参数距离通常不合理，更合理的是比较它们的输出函数、预测分布或 loss landscape 中的连接路径。

## Q10：收敛性证明

建议不要把证明写太长，否则会淹没主线。你可以这样组织：

### 梯度下降法

假设 (f) 是凸且 (L)-smooth，取：

[
x_{k+1}=x_k-\frac1L\nabla f(x_k).
]

由光滑性：

[
f(x_{k+1})
\le
f(x_k)
-\frac1{2L}|\nabla f(x_k)|^2.
]

结合凸性和望远镜求和，可以得到：

[
f(x_k)-f(x^*)
\le
\frac{L|x_0-x^*|^2}{2k}.
]

所以凸情形下 GD 是：

[
O(1/k).
]

若 (f) 进一步是 (\mu)-强凸，则：

[
f(x_k)-f(x^*)
\le
\left(1-\frac{\mu}{L}\right)^k
[f(x_0)-f(x^*)].
]

所以强凸情形下是线性收敛。

### Nesterov 动量加速

凸情形：

[
f(x_k)-f(x^*)
\le
\frac{2L|x_0-x^*|^2}{(k+1)^2}.
]

即：

[
O(1/k^2).
]

强凸情形：

[
f(x_k)-f(x^*)
\le
C\left(1-\sqrt{\frac{\mu}{L}}\right)^k.
]

这里不要硬写完整复杂证明，可以写：

> Nesterov 加速法的证明通常通过构造估计序列或 Lyapunov 势函数完成。势函数同时包含函数误差项和距离误差项，通过选择合适的动量参数，使势函数单调下降，从而得到 (O(1/k^2)) 或依赖 (\sqrt{\kappa}) 的线性收敛率。

这样严谨且不臃肿。
